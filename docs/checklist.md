# Build Checklist — Read4ever

## Build Preferences

- **Build mode:** Step-by-step
- **Comprehension checks:** Yes — after each item, agent asks a quick question to confirm the pieces are connecting
- **Git:** Commit after each item with message: "Complete step N: [title]"
- **Verification:** Yes — learner runs the app after each item and confirms acceptance criteria before moving on
- **Check-in cadence:** Learning-driven — agent explains key decisions, tradeoffs, and patterns as each item is built
